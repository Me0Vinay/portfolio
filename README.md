# Professional Portfolio - Vinay Kr

A modern, responsive portfolio website featuring a BI & Data Engineering roadmap (Gantt Chart), project showcases, and interactive technical skills display.

## Features
- **Interactive Career Roadmap**: A custom Gantt chart visualizing professional work experience and core technical skills.
- **Project Showcase**: Categorized grid of featured data and BI projects with direct links.
- **Responsive Design**: Optimized for all screen sizes with smooth glassmorphism UI elements.
- **Custom Alert Library**: Showcasing `easycherry`, a Python library for automated alerting.

## Tech Stack
- HTML5 / CSS3 (Vanilla)
- Modern JavaScript (ES6+)
- Design: Glassmorphism / Dark Mode
- Data Visualization: Custom CSS-based Gantt Chart
